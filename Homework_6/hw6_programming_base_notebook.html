Kernel Regression¶

Given a training dataset $\{x_i, y_i\}_{i=1}^n$, kernel regression approximates the unknown nolinear relation between $x$ and $y$ with a function of form $$ y\approx f(x;w) = \sum_{i=1}^n w_i k(x, x_i), $$ where $k(x, x')$ is a positive definite kernel specified by the users, and ${w_i}$ is a set of weights. We will use the simple Gaussian radius basis funciton (RBF) kernel, $$ k(x, x') = exp(-\frac{||x-x'||^2}{2h^2}), $$ where $h$ is a bandwith parameter.

def rbf_kernel(X, Xp, h):

Step 3. The median trick for bandwith¶

The choice of the bandwidth $h$ A common way to set the bandwith $h$ in practice is the so called median trick, which sets $h$ to be the median of the pairwise distance on the training data, that is $$ h_{med} = median(\{||x_i - x_j||: i\neq j, ~~~~~~i,j =1, ..., n\}). $$

from scipy.spatial import distance

def median_distance(X):
    # X: n*1 matrix
     
    #TODO: Calculate the median of the pairwise distance of $X$ below 
    #(hint: use '[dist[i, j] for i in range(len(X)) for j in range(len(X)) if i != j]' to remove the diagonal terms; use np.median)
    dist = distance.cdist(X, X, 'euclidean')
    valuess = [dist[i, j] for i in range(len(X)) for j in range(len(X)) if i != j]

    h = np.median(valuess)
    
    return h

Step 4. Kernel regression¶

The weights ${w_i}$ are estimated by minimizing a regularized mean square error: $$ \min_w {\left ( \sum_{i=1}^n (y_i - f(x_i; w))^2 \right) + \beta w^\top K w }, $$ where $w$ is the column vector formed by $w=[w_i]_{i=1}^n$ and K is the kernel matrix.

def kernel_regression_fitting(xTrain, yTrain, h, beta=1):
    # X: input data, numpy array, n*1
    # Y: input labels, numpy array, n*1
    
     
    # TODO: calculate W below (it is a n*1 matrix)
    kernel = rbf_kernel(xTrain, xTrain, h)
    n = kernel.shape[0]

    W = np.linalg.solve(kernel + beta * np.eye(n), yTrain)
  
    return W

def kernel_regression_fit_and_predict(xTrain, yTrain, xTest, h, beta):
    
    #fitting on the training data 
    W = kernel_regression_fitting(xTrain, yTrain, h, beta)
    
    # computing the kernel matrix between xTrain and xTest
    K_xTrain_xTest = rbf_kernel(xTrain, xTest, h)
   
    # predict the label of xTest
    yPred = np.dot( K_xTrain_xTest.T, W)
    return yPred

best_beta, best_coff = 1., 1.
best_mse = 1e8
for beta in [0.1, 1]:
    for coff in [0.1, 1., 10.]:
        # 5-fold cross validation
        max_fold = 5
        mse = []
        for i in range(max_fold):
            
            ##TODO: calculate the index of the training/testing partition within 5 fold CV.
            # (hint: set trnIdx to be these index with idx%max_fold!=i, and testIdx with idx%max_fold==i)

            idx = np.arange(len(xTrain))
            trnIdx = idx[idx % max_fold != i]
            testIdx = idx[idx % max_fold == i]
            
            i_xTrain, i_yTrain = xTrain[trnIdx], yTrain[trnIdx]
            i_xValid, i_yValid = xTrain[testIdx], yTrain[testIdx]
            
            
            ##TODO: run kernel regression on (i_xTrain, i_yTrain) and calculate the mean square error on (i_xValid, i_yValid)
            h = coff * median_distance(i_xTrain)  
            i_yPred = kernel_regression_fit_and_predict(i_xTrain, i_yTrain, i_xValid, h, beta)           
            mse.append((i_yValid - i_yPred)**2)
            
        mse = np.mean(mse)
        
        # keep track of the combination with the best MSE
        if mse < best_mse:
            best_beta, best_coff = beta, coff
            best_mse = mse
